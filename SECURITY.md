# Security Policy

## Supported Versions

Below are the versions that we continue to provide support for. If you have a version with an ❌ please do not open an issue or contact any of the maintainers to provide support!

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | ✅                 |

## Reporting a Vulnerability

You can report a vulnerability by opening an issue in this repo. We will always try our best to keep everything up-to-date and secure!

> *Once a vulnerability is spotted we will try to push an update as fast as possible!*
